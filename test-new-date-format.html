New Date Format Test

This tests the new text-based date input that should fix iOS app compatibility issues.

Test Date Input

Format: YYYY-MM-DD (e.g., 2025-08-25)
Format: HH:MM (24-hour format, e.g., 06:00, 17:30)

Expected iOS App Format

The iOS app expects dates in this exact format:

YYYY-MM-DDTHH:MM:SS.sssZ

Example: 2025-08-25T00:00:00.000Z

Note: The time is always set to 00:00:00.000Z (midnight UTC) for maximum compatibility.
